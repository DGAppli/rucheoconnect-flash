Mise à jour ruche C3 2.14

⚠️ Navigateur non compatible. Cette page nécessite Google Chrome ou Microsoft Edge (version récente). Firefox et Safari ne supportent pas l'API Serial.
💻 Pilotes USB requis (Windows uniquement) Installez les deux pilotes — ils coexistent sans problème.
⬇️ Pilote CH340 (le plus courant) ⬇️ Pilote CP2102 (Silicon Labs)
🐧 Linux détecté — aucun pilote requis Linux reconnaît nativement les modules ESP32. Utilisez directement Google Chrome ou Microsoft Edge pour flasher.
1
Sortez le module C3 de la ruche Débranchez et retirez le module ESP32-C3.
2
Branchez le câble USB sur le C3 Connectez le câble USB directement sur le port micro USB du module C3.
3
Cliquez sur « Flasher la ruche » Sélectionnez le port série dans la fenêtre qui s'ouvre, puis attendez la fin (~2 min). Rebranchez ensuite le module dans la ruche.
🚨 NE COCHEZ PAS la case "Erase device" — cela effacerait la calibration et tous les réglages de la ruche.
