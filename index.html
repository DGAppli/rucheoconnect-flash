Mise à jour passerelle GW 1.0.60

⚠️ Navigateur non compatible. Cette page nécessite Google Chrome ou Microsoft Edge (version récente). Firefox et Safari ne supportent pas l'API Serial.
1
Sortez le module ESP32 du boîtier Tirez doucement sur le module ESP32 pour l'extraire du boîtier (voir photo ci-dessous).

📷 Tirez sur le module ESP32 pour le sortir du boîtier

2
Branchez le câble USB sur l'ESP32 Connectez le câble USB-C directement sur le module ESP32 sorti du boîtier.
3
Passez l'ESP32 en mode flash Maintenez le bouton BOOT enfoncé, appuyez brièvement sur RESET, puis relâchez BOOT. La LED s'éteint : l'ESP32 est prêt.
4
Cliquez sur « Flasher la passerelle » Sélectionnez le port série dans la fenêtre, puis attendez la fin (~2 min). Remettez ensuite l'ESP32 dans le boîtier.
