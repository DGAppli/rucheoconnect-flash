Mise à jour passerelle GW 1.0.60x

⚠️ Navigateur non compatible. Cette page nécessite Google Chrome ou Microsoft Edge (version récente). Firefox et Safari ne supportent pas l'API Serial.
1
Sortez le module ESP32 du boîtier Tirez doucement sur le module ESP32 pour l'extraire du boîtier (voir photo ci-dessous).

📷 Tirez sur le module ESP32 pour le sortir du boîtier

2
Branchez le câble USB sur l'ESP32 Connectez le câble USB directement sur le port USB-C du module ESP32 sorti du boîtier.
3
Cliquez sur « Flasher la passerelle » Une fenêtre apparaît pour sélectionner le port série. Choisissez le port correspondant à votre passerelle (généralement le seul disponible).
4
Attendez la fin de la mise à jour La barre de progression indique l'avancement. Ne débranchez pas le câble pendant l'opération (~2 min). Remettez ensuite l'ESP32 dans le boîtier.
